メトリックスレッドはピッチ (mm per thread) で指定され、インペリアル (UN) スレッドはthread per inch (TPI) で指定されます。変換: TPI = 25.4/ピッチ (mm)。M10 × 1.5は、5/16〜18 UNCに最も近い1.5mm = 16.9TPIのピッチを有する。M12 × 1.75 = 14.5TPI =1/2-13 UNC。スレッドミリングの場合、これらのシステム間の変換により、コストのかかるツールパスエラーが正しく回避されます。

直径の比較: メトリックとインペリアルスレッド

メトリックねじ径は、公称OD (mm) として指定される。一般的な変換: M6約1/4インチ (6.35mm対6.0mm) 、M8約5/16インチ (7.94mm対8.0mm) 、M10約3/8インチ (9.53mm対10.0mm) 、M12約1/2インチ (12.7mm対12.0mm)。近い間、これらは交換可能ではありません-常に正しいファスナーシステムを使用してください。

実用的なスレッド: メトリック対インペリアルを使用するとき

最新のCNCマシンは、両方のシステムを均等に切断できます。お客様の図面またはサプライチェーンのファスナー標準で指定されたシステムを使用します。メートル法のドリルとタップが世界中でより広く利用可能であるため、既存の帝国の設計をメートル法に変換すると、工具の在庫を減らすことができます。ただし、特殊なスレッドゲージのコストとリードタイムを考慮する必要があります。

よくある質問

インペリアルファスナーの代わりにメトリックファスナーを使用できますか?お勧めしません。直径0.3〜0.7mmの違いにより、スレッドの噛み合いや干渉が不十分になり、関節の故障を引き起こす可能性があります。M10を3/8-16に置き換えると、負荷がかかっているスレッドが剥がれる可能性があります。指定されたスレッドシステムに常に部品を機械化します。

スレッドクラス (6 H対2 B) を変換するにはどうすればよいですか?ISO 6 Hメートルトレランスは、中適合アプリケーションのUN 2 Bクラスに近似します。精密フィットの場合は4 H6 H (メトリック) 約3 B (UN) 、ルーズフィットの場合は7 H (メトリック) 約1 B (UN)。これらは近似です-重要なジョイントの場合は、設計当局によって指定された標準を使用します。

メートル法とインペリアルドリルはスレッドに互換性がありますか?メトリックタップドリル (たとえば、M10 × 1.5の場合は8.5mm) には、正確なインペリアル同等物はありません。必要に応じて、利用可能な最も近いドリルサイズを使用しますが、好ましい方法は、切断されるスレッドシステムに指定されたタップドリルを使用することです。たとえば、8.3mmメートルドリルではなく、3/8-16 UNCには「Q」ドリル (0.332インチ) を使用します。

Can I substitute a metric fastener for an imperial one? Not recommended. The 0.3–0.7 mm diameter differences create insufficient thread engagement or interference, which can cause joint failure. Substituting M10 for 3/8-16 may strip threads under load. Always machine parts to the specified thread system.

How do I convert thread class (6H vs. 2B)? ISO 6H metric tolerance approximates to UN 2B class for medium-fit applications. 4H6H (metric) ≈ 3B (UN) for precision fits, and 7H (metric) ≈ 1B (UN) for loose fits. These are approximations — for critical joints, use the standard specified by the design authority.

Are metric and imperial drills compatible for threading? Metric tap drills (e.g., 8.5 mm for M10×1.5) don't have exact imperial equivalents. Use the closest available drill size if needed, but preferred practice is to use the specified tap drill for the thread system being cut. For example, use a 'Q' drill (0.332 inch) for 3/8-16 UNC, not an 8.3 mm metric drill.
