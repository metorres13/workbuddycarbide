NPT (National Pipe Taper) スレッドは、直径で測定された1:16テーパー (0.75インチ/フィート) を持っています。糸の角度は60 ° で、頂上と根が切り詰められています。テーパーは、ゲージングノッチのピッチ直径がスレッドのシーリング能力を決定することを意味します。NPTスレッドは、スレッド自体の変形によってシールされます-干渉フィットは、シーラントで適切に組み立てられたときに耐圧ジョイントを作成します。

加工用のNPTスレッド寸法の計算

NPTスレッドをプログラミングするための重要な寸法: スレッドの深さ = 0.8 × 切り捨てられたプロファイルのピッチ、テーパー角度 = 1.79 ° が含まれています (1辺あたり0.895 ° の半角)。1/2 NPTスレッドの場合: ピッチ = 14 TPI (0.0714インチ) 、スレッドの深さ = 0.0571インチ、スレッドはハンドタイトエンゲージメントの場合は約0.54インチ、レンチタイトの場合は0.25インチ伸びます。

NPT対NPS対BSPT: 違いは何ですか?

NPT (National Pipe Taper) とBSPT (British Standard Pipe Taper) は、外観が似ているにもかかわらず、互換性がありません。NPTは60 ° のねじ角度を有し、BSPTは55 ° を有する。BSPTはまた、異なるピッチシリーズを使用する。NPS (National Pipe Straight) には、スレッド干渉ではなくシーリングワッシャーに依存する平行スレッドがあります。パイプの糸を切る前に、常に標準を確認してください。

よくある質問

NPT継手にどのくらいのスレッドシーラントを使用する必要がありますか?シーラント (パイプドープまたはPTFEテープ) は、端から1〜2本の糸から始めて、雄の糸にのみ塗布します。糸の根を埋めますが、パイプシステムに入る可能性のある余分なシーラントは避けてください。PTFEテープの場合は、時計回りに2〜3回巻き (パイプの端を見るとき) 、50% オーバーラップします。

NPT接続で許容できるスレッドの長さはどのくらいですか?この規格では、ハンドタイトエンゲージメント (L1) とレンチタイトメイク (L3) を指定しています。1/2 NPTの場合、L1 = 0.320インチ、L3 = 0.533インチ。最大スレッド長はL1L3である。これを超えると、スレッドが深くカットされすぎているか、テーパーが正しくありません。

NPTスレッドゲージを検査に使用できますか?はい。オスの糸にはL1 (ハンドタイト) リングゲージを使用し、メスの糸にはプラグゲージを使用します。スレッドは手でゲージに入り、ゲージノッチの ± 1ターン以内に停止する必要があります。最終検査には6ステップのL2ゲージを使用して、スレッドが深く切断されすぎていないことを確認します。

How much thread sealant should I use on NPT fittings? Apply sealant (pipe dope or PTFE tape) to the male threads only, starting 1–2 threads from the end. Fill the thread roots but avoid excess sealant that could enter the pipe system. For PTFE tape, wrap 2–3 turns clockwise (when viewing the pipe end) with 50% overlap.

What is the acceptable thread length for NPT connections? The standard specifies hand-tight engagement (L1) and wrench-tight make-up (L3). For 1/2 NPT, L1 = 0.320 inches and L3 = 0.533 inches. Maximum thread length is L1 + L3. Exceeding this indicates the thread is cut too deep or the taper is incorrect.

Can I use NPT thread gages for inspection? Yes. Use L1 (hand-tight) ring gages for male threads and plug gages for female threads. The thread should enter the gage by hand and stop within ±1 turn of the gage notch. Use a 6-step L2 gage for final inspection to verify the thread is not cut too deep.
